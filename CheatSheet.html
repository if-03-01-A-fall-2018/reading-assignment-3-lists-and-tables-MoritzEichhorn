Web Server

What is it?
is a program that servers the files that form Web pages to users via HTTP
What does it?
stores, processes and delivers web pages to clients
Storage
The resource is typically a real file on the server's secondary storage, but it depends on how it is implemented

Web Client

What is it?
It is a software application for accessing information on the WWW
What does it?
Fetches information rescources and displays them on a user's device

Html for Lists

Unordered List
Start tag: <ul>
List item: <li>
Ordered List
Start tag: <ol>
List item: <li>
Description List
Start tag: <dl>
Description Term: <dt>
Description Data: <dd>

Css for Lists

Different List Item Markers
list-style-type: circle/square/upper-roman/lower-alpha;
Image as List Item Marker
list-style-image: url('url.png');
Position List Item Markers
list-style-position: outside/inside;
Remove Default Setting
list-style-type: none;
margin: 0;
padding: 0;

Html for Tables

Defining an HTML Table
Start tag: <table>
Table row: <tr>
Table header: <th>
Table data: <td>
thead, tbody, tfoot!!!
Cells that span many Columns/Rows
<th col/rowspan="2">
Caption
Start tag: <caption>

Css for Tables

Table Borders
table, th,td { border: 1px solid black; }
Collapse Table Borders
border-collapse: collapse;
Table Width and Height
table { width: 100%; }
th { height: 50px; }
Striped Tables
tr:nth-child(even/odd) { background-color: #f2f2f2; }
Border Style
Property: border-style
Values: dotted, dashed, solid, double, groove, ridge, inset, outset, none, hidden
Border Width
Property: border-width
Values: (one) 5px/medium, (four) 2px 10px 4px 20px
Border Color
property: border-color
values: as usual
Individual Sides
Property: border-top/right/bottom/left-style
or border-style: top right bottom left;
Shorthand Property
border: width style color;
Rounded Borders
Property: border-radius
